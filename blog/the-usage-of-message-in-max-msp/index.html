浅谈 Max/MSP 中 message 的用途

05/05/2018 15:05
  • message box 的功能和用途：
  1. 点击时生成信息
  2. 当一个信息被收到时，生成新的信息
  3. 储存并创建混合类型（文本、数字和列表型）的数据
  4. 展示输入的信息
  5. 用在其他对象上的指令（比如 open message、reset message）

  • 变量的占位符：$1、$2 等

  • 参数的用法：

    • append 和 prepend，在输入的信息后面或者前面加上特定的信息段

  • 使用 message 作为指令：

    • 改变下一个对象的外观，例如：bgcolor 0.1 0. 0. 1.
